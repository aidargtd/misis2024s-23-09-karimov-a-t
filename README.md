# Navigator
В данном репозитории в директории prj.labs представлены реализации следующих структур данных, написанных на языке c++:
  - BinarySearchTree
  - CustomContainer
  - DoubleListQueue
  - HashTable
  - StackArrT
  - StackLstT
  - complex
  - dynarr
  - iStack
  - rational
Кроме того, в директории prj.test лежат тесты к каждой библиотеке.
